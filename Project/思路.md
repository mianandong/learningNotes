# Web页面快速搭建教学工具页面
### 概述
```
在Web端，通过引用teachingtool-sdk，可以快速的搭建一个带有各种教学工具（如圆规，直尺）的页面。
teachingtool-sdk提供基础的面板及通用操作（如工具的框选和移动），但并不实现教学工具相关组件，
而是制定规则，提供基础组件，让更多的第三方（许许多多的teachingtool-component-sdk）具体实现教学工具。
```
## Web端应用 teachingtool-web
